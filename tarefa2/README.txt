# Projeto: Hierarquia de Formas Geométricas em C

## Dependências:
- Compilador `gcc` (ou compatível).

## Uso do Makefile:
- `make`: Compila o projeto e gera o executável.
- `make run`: Compila e executa o programa.
- `make clean`: Remove os arquivos `.o` e o executável.

### Observações:
- No Windows, o comando `clean` usa `del` para remover arquivos com a extensão `.exe`. Ajuste conforme necessário para seu sistema.
